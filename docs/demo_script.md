# Demo Script

Target duration: 3 minutes.

## Opening

This is a local prototype. It does not call real AI APIs or external systems. The purpose is to show the AI workflow: data in, analysis, recommendation, impact report.

## Flow

1. Select the topic from the top control.
2. Load the sample scenario.
3. Show the dashboard metrics.
4. Explain the AI analysis cards.
5. Review the recommendation panel.
6. End on the impact report.

## Key Message

The demo should make one point clear: the product is not a static dashboard. It converts data into an AI-supported decision workflow.
